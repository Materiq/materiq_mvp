# Materiq

AI-powered procurement and quotation copilot for interior designers and architects.

## Overview
Materiq automates:
- BOQ generation
- quotation workflows
- material estimation
- procurement coordination
- vendor discovery workflows

The goal is to reduce manual effort in interior project management and streamline commerce-oriented procurement operations.

## Planned Features
- AI-generated BOQs
- Automated quotation PDFs
- Vendor coordination dashboard
- Material cost estimation
- Procurement workflow automation
- Project tracking

## Tech Stack
- React
- Supabase
- OpenAI APIs
- Vercel

## Status
Currently in MVP development phase.
